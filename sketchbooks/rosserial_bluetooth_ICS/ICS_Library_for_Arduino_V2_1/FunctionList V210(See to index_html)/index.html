IcsClassの概要

このライブラリは近藤科学製ロボット用サーボ(KRSシリーズ)を動かすためのライブラリです。
ICS3.5およびICS3.6に対応しています。
現行では、Arduino製品で使えるように設計しています。
使い方および詳細は、下記弊社HPをご覧ください。
http://kondo-robot.com/
不具合等ありましたら、弊社HPを参照にご連絡ください。
